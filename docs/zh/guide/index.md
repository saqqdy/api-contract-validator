# 介绍

API Contract Validator 帮助你在 OpenAPI/Swagger 规范与实现代码之间的**破坏性变更**到达生产前发现它们。

## 问题

API 规范与实现代码会悄悄漂移：
- 代码中缺失的端点
- 额外未记录的端点
- Schema 不匹配
- 参数类型变更

## 解决方案

三层验证：
1. **规范解析** — 解析 OpenAPI/Swagger 为标准化端点
2. **代码分析** — 从 Express/Fastify/Koa/NestJS 提取路由
3. **漂移检测** — 比较规范 vs 代码，按严重程度分类

## 严重程度

| 级别 | 含义 |
|------|------|
| 🔴 破坏性 | 会破坏客户端 |
| 🟡 警告 | 可能引起问题 |
| 🟢 信息 | 非破坏性 |
